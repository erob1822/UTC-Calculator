# UTC Calculator

A lightweight, offline-capable UTC time calculator built for in-flight use. Designed for quick, accurate 24-hour time arithmetic in high-stress, low-light environments.

## Features

- **Live UTC Clock** — Displays the current UTC time in large, easy-to-read format.
- **Clock Calibration** — Manually set the displayed UTC time if the system clock is inaccurate, with a one-click revert to system time.
- **Time Calculator** — Enter a starting UTC time, then apply one or more hour/minute/second modifiers (+ or −) to compute a resultant UTC time.
- **Note Current** — Instantly capture the current UTC time as your starting time.
- **Real-Time Updates** — The resultant time updates immediately as inputs change.
- **Fully Offline** — All assets, including the RPL logo, are embedded directly in the HTML. No network connection required.
- **Mobile-Friendly** — Optimized for small screens with a dark theme suited for cockpit and low-light use.

## Usage

Open `index.html` in any modern web browser. No build step, server, or internet connection needed.

Or: this program is live at https://erob1822.github.io/UTC-Calculator/
